# T020 — T020 - Add task edit and delete interactions in the UI

**Source**: GitHub Issue #21

## Description

## Goal

Allow users to edit and delete existing tasks from the UI.

## Depends on

- T017 - Implement task list and quick create UI

## Scope

- Add edit form or inline edit interaction.
- Add delete action with confirmation.
- Update local UI state after successful mutation.
- Show validation and API errors.

## Acceptance criteria

- Users can edit task title, description, priority, status, category, and planned dates.
- Users can delete a task after confirmation.
- UI updates correctly after edit/delete.
- Tests cover successful edit and delete flows.
