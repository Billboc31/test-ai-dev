# T018 — T018 - Add monthly calendar view for tasks

**Source**: GitHub Issue #19

## Description

## Goal

Add a monthly calendar view for planned tasks.

## Depends on

- T016 - Add frontend API client and task data hooks

## Scope

- Integrate `react-big-calendar`.
- Display tasks with planned dates on a monthly calendar.
- Add basic navigation between months.
- Keep styling simple and responsive.

## Acceptance criteria

- Tasks with planned dates appear on the calendar.
- Month navigation works.
- Tasks without planned dates do not break the calendar.
- Basic frontend tests cover calendar rendering.
